`Win11Optimizer` — это инструмент с открытым исходным кодом, написанный на C# (.NET 8). Его главная цель — предоставить пользователям удобный консольный интерфейс для тонкой настройки операционной системы, отключения лишних функций и повышения общей производительности ПК (в том числе в играх).

Ключевое отличие этого проекта от аналогичных скриптов — **абсолютная безопасность**. Любое изменение в системе может быть отменено в один клик.

## 🛡 Безопасность и Откат изменений

Каждое действие программы логируется и защищается с помощью встроенного модуля `BackupManager`.
Перед применением любой оптимизации программа **автоматически**:
1. Создает системную точку восстановления (System Restore Point).
2. Делает экспорт изменяемой ветки реестра в файл формата `.reg`.

В случае возникновения проблем вы можете выбрать опцию "Откат" прямо в главном меню программы.

## 🚀 Возможности и Модули

Проект построен по модульной архитектуре. Каждая оптимизация представляет собой независимый плагин.

### 🌐 Сеть (Network)
* **Nagle's Algorithm (`TCPNoDelay`)**: Отключение искусственной задержки объединения мелких пакетов. Радикально снижает пинг в соревновательных онлайн-играх.
* **Network Throttling**: Снятие ограничений службы MMCSS для приоритизации процессорного времени под игры.
* **TCP/IP Stack**: Тонкая настройка параметров сети (TTL, Window Auto-Tuning) для минимизации потерь и задержек.

### 🎮 Игры (Gaming)
* **Game Mode**: Принудительное включение игрового режима Windows 11.
* **GPU Scheduling**: Активация аппаратного планировщика графического процессора для уменьшения `Input Lag`.
* **Game DVR**: Полное отключение фоновой записи экрана и оверлея Xbox Game Bar для высвобождения ресурсов GPU и CPU.

### 🔐 Приватность (Privacy)
* **Telemetry**: Ограничение сбора диагностических данных Microsoft до минимального уровня `Security`.
* **Privacy Tweaks**: Отключение рекламного идентификатора (Advertising ID), отключение синхронизации истории активности пользователя (Timeline) и запрет на сбор статистики запускаемых приложений.

### ⚙️ Система (System)
* **Services**: Безопасное отключение ресурсоемких фоновых служб, таких как `SysMain` (рекомендуется для SSD), `DiagTrack` и других процессов телеметрии.
* **Visual Effects**: Отключение тяжелых анимаций окон и прозрачности `Mica` для повышения отзывчивости интерфейса ОС.
* **Power Plan**: Перевод системы в режим электропитания "Высокая производительность".

## 📦 Установка и Использование

Вам не нужно устанавливать .NET SDK. Программа поставляется в виде независимого портативного `.exe` файла.

1. Перейдите в раздел [Releases](../../releases) и скачайте актуальный `Win11Optimization_Portable.zip`.
2. Распакуйте архив в любую директорию.
3. Кликните правой кнопкой мыши по `Win11Optimization.CLI.exe` и выберите **"Запуск от имени Администратора"** *(это обязательно для работы с системными службами и реестром)*.
4. Выберите нужные модули в интерактивном меню с помощью пробела (Space) и нажмите `Enter`.

## 🛠 Сборка из исходного кода (Для разработчиков)

Проект использует C# 12 и платформу .NET 8.0.

```powershell
# Клонирование репозитория
git clone https://github.com/CTPAX4OK/Win11Optimization.git
cd Win11Optimization

# Сборка и запуск локально
dotnet run --project src\Win11Optimization.CLI

# Сборка портативного (Single-File) исполняемого файла
dotnet publish src\Win11Optimization.CLI\Win11Optimization.CLI.csproj -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o release_build
```

## 🤝 Контрибьютинг

Архитектура проекта поддерживает внедрение зависимостей (Dependency Injection). Добавление новых функций максимально упрощено:
1. Создайте класс, реализующий интерфейс `IOptimization`.
2. Опишите метаданные в свойстве `Info`.
3. Реализуйте методы проверки (`IsAppliedAsync`), применения (`ApplyAsync`) и отката (`RollbackAsync`). Для работы с реестром используйте безопасную обертку `RegistryHelper`.
4. Зарегистрируйте ваш класс в `ServiceCollectionExtensions.cs`.
Главное меню автоматически подхватит и отобразит ваш новый модуль.
